Use SeisCloud with SeisClient

SeisClient Tutorial

SeisClient is a Python package to request accurate 3D Greens’ function and synthetic waveform from SeisCloud which is a data server hosting pre-computed receiver-side 3D Greens’ function database.

SeisClient can be utilized to request Greens’ function and synthetic waveform from point sources, multiple-point sources, and finite faults for various applications such as earthquake source characterization, induced seismicity monitoring, and Earthquake Early Warning (EEW).
